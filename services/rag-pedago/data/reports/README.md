# data/reports/

Rôle : rapports Codex versionnés et rapports runtime locaux.

Peut contenir :

- `codex_lot_*.md` versionnés ;
- rapports runtime readiness, coverage, gate, review package, controlled import.

Interdit :

- secrets ;
- exports de documents sources ;
- données personnelles élèves.

Les rapports runtime sont ignorés par Git. Les rapports Codex sont conservés.
