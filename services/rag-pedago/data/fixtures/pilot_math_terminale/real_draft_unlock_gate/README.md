# Fixtures gate combiné human unlock + real draft metadata

Ces fixtures sont synthétiques. Elles testent uniquement la cohérence entre une
autorisation humaine metadata-only et des items metadata-only candidats.

Elles ne contiennent :

- aucun document source ;
- aucun PDF, DOCX, PPTX ou XLSX ;
- aucun chemin de fichier local réel ;
- aucun secret réel ;
- aucun manifest réel prêt pour import.

Les `source_uri` valides restent en `synthetic://`.
