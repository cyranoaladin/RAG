# Fixtures autorisation humaine

Ces fixtures sont synthetiques et servent uniquement a tester le verrou humain
du lot 15G.

Elles ne contiennent :

- aucun document source ;
- aucun PDF, DOCX, PPTX ou XLSX ;
- aucun chemin `file://` reel ;
- aucun secret reel ;
- aucun brouillon de manifest pret pour import.

La fixture dont le nom contient `secret_marker` est volontairement invalide et
sert a verifier le refus d'un marqueur sensible synthetique.
